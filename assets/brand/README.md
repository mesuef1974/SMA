## فكرة الشعار — "المربّع الميم"

حرف **م** العربي بحلقة مرسومة كمربّع هندسي تامّ. إزدواجية دلالية خالصة:
- **للعين العربية:** حرف مِيم (أوّل حروف: محلل، معلّم، متعلّم، معادلة، مدرسة).
- **للعقل الرياضي:** المربّع — أساس الجذر، التربيع، الشبكة الإحداثية، الهندسة.

شكلان فقط: مربّع مُدوَّر الزوايا + خط عمودي قصير. لا تفاصيل زائدة. يشتغل من 16px إلى لافتة. يعمل بالأبيض والأسود كما يعمل بالألوان — وهذا هو اختبار Paul Rand للشعار الجيّد.

## البدء السريع — تطبيق الهوية

1. **افتح `brand-book.html`** في المتصفح لرؤية كامل النظام بصرياً.
2. **انسخ `tokens.css`** إلى `app/src/app/globals.css` (استبدل كتلتي `:root` و `.dark`).
3. **انقل `favicon.svg`** إلى `app/public/` وحدِّث `app/src/app/layout.tsx`:
   ```tsx
   export const metadata = {
     icons: { icon: '/favicon.svg' },
   };
   ```
4. **استبدل placeholder "م"** في:
   - `app/src/components/dashboard/dashboard-sidebar.tsx` (السطر 79–80)
   - `app/src/app/[locale]/login/page.tsx` (السطر 21–23)
   
   بمكوّن React يستورد الـ SVG:
   ```tsx
   import Mark from '@/../../brand/logo/mark.svg';
   ```
   أو ضع SVG مباشرة inline.

5. **أضف IBM Plex Sans Arabic** في `app/src/app/layout.tsx`:
   ```tsx
   import { IBM_Plex_Sans_Arabic, IBM_Plex_Mono } from 'next/font/google';
   
   const arabic = IBM_Plex_Sans_Arabic({
     variable: '--font-sans',
     subsets: ['arabic'],
     weight: ['300', '400', '500', '600', '700'],
   });
   ```

## فلسفة التصميم

> "التصميم يحلّ مشكلة تجارية، والفنّ يعبّر عن رأي." — Paul Rand

كل قرار في هذا النظام له مبرّر، ليس ذوقاً:

- **لماذا الأنديغو العميق؟** لأنه يعكس الرصانة العلمية والثقة — مقابل الرمادي العديم الشخصية في shadcn الافتراضي.
- **لماذا حرف م مربّع؟** لأنه يحلّ معادلة "كيف نُميّز منصّة عربية للرياضيات بصرياً؟" في شكلين.
- **لماذا IBM Plex؟** لأنه يوازن الدقّة الهندسية (مناسبة للرياضيات) مع دفء إنساني كافٍ ليتنفّس النصّ العربي.
- **لماذا ثلاثة ألوان فقط؟** لأن العلامات القوية لها لوحة مركّزة. النجم، القمر، السهل — كل لون يحمل معنى ودوراً.
