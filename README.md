## 📋 Workflow

1. Create a \eature/*\ branch from \develop\
2. Work on your feature with meaningful commits
3. Open a Pull Request to \develop\
4. After review, merge to \develop\
5. When ready for release, create \elease/*\ from \develop\
6. Merge \elease/*\ to both \main\ and \develop\
7. Tag the release on \main\
